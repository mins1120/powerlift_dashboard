# 파워리프팅 대쉬보드 및 사용자 역량 분석

## 개요
파워리프팅 선수들의 3대운동 역량 데이터를 분석하여 사용자의 운동 역량 향상에 도움이 되는 정보를 제공하는 앱을 제작하였습니다. 사용자는 자신의 운동 역량 수준과 목표로 하는 체급을 달성하기 위한 계산기등을 제공받을 수 있습니다.

## 기능

1. 성별 분포
2. 체급별 최고 성적
3. 연령대별 성적 분포
4. 장비별 성적 분석
5. 성별 및 체급별 Wiklks 점수 평균 분포
6. Wilks 점수에 따른 성능 비교
7. 개인 성적 분석 및 조언
8. 개인 데이터 제공 동의
9. 기초대사량 계산기
